# 꿀벌백과사전

React로 만든 꿀벌 생태 백과사전 웹앱입니다. 꿀벌의 종류, 군체 안의 역할, 행동, 생태, 보호 방법을 검색하고 분류별로 탐색할 수 있습니다.

## 주요 기능

- 꿀벌 항목 검색
- 종류, 역할, 행동, 생태, 보호 카테고리 필터
- 항목별 상세 설명과 핵심 사실 표시
- 선택 항목에 따른 미니 퀴즈
- 꿀벌 보호 실천 팁 소개

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드 방법

```bash
npm run build
```
